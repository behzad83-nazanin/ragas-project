# 🚀 RAGAS Evaluation Project

A Retrieval-Augmented Generation (RAG) evaluation project built with Python and RAGAS.

## 📌 Features

- Evaluate RAG pipelines
- Generate evaluation metrics
- CSV-based dataset evaluation
- Easy-to-use Streamlit interface
- Modular project structure

## 🛠 Tech Stack

- Python
- RAGAS
- LangChain
- OpenAI
- Pandas
- Streamlit

## 📂 Project Structure

```
ragas-project/
│── src/
│── app.py
│── requirements.txt
│── evaluation_data.csv
│── evaluation_data_10_samples.csv
│── .env
```

## ⚙️ Installation

```bash
git clone https://github.com/behzad83-nazanin/ragas-project.git

cd ragas-project

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt
```

## ▶️ Run

```bash
streamlit run app.py
```
